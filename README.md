# Smart Healthcare IoT Simulation (Colab Project)

## Overview
This project was created using Google Colab to simulate and analyze healthcare IoT patient data.

## Dataset
Includes synthetic patient data:
- Heart Rate
- Blood Pressure
- Temperature
- Oxygen Level
- Steps

## Tools Used
- Google Colab
- Python
- Pandas
- Matplotlib
- Seaborn
